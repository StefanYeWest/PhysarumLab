# Документация Physarum Lab

Этот каталог содержит проектную документацию и материалы для отчёта по практике.

- [`architecture.md`](./architecture.md) — архитектура, 4 основных класса, жизненный цикл тика, UML-диаграммы.
- [`testing-plan.md`](./testing-plan.md) — план тестирования, таблица ручных тестов, список unit-тестов и результаты.
- [`report-materials.md`](./report-materials.md) — готовые текстовые блоки для разделов отчёта (цель, задачи, описание модели и алгоритма, выводы экспериментов).

Корневой [`README.md`](../README.md) содержит инструкцию запуска и пользовательское описание.

## Как снять скриншоты для отчёта

1. `npm run dev`, открыть `http://localhost:5173`.
2. Загрузить сценарий «Выставочная демонстрация», нажать «Старт».
3. Сделать скриншоты: стартовое состояние, формирование сети, сравнение с A*, перестройка после препятствия, панель метрик.
4. Сохранить в `docs/screenshots/` (создайте каталог) и приложить в «Приложение А» отчёта.
